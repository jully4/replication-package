# replication-package
Replication package for peer review
# Pacote de Replicação

Este repositório contém as planilhas utilizadas durante a condução da Revisão da Literatura Cinzenta apresentada no manuscrito submetido para avaliação.

As planilhas reúnem os dados extraídos das publicações analisadas e serviram de base para a organização, classificação e análise dos resultados apresentados no estudo.

## Conteúdo do repositório

O pasta DADOS é composta pelas seguintes planilhas:

- **Artigos Selecionados:** contém os 59 artigos recuperados na busca inicial, bem como a indicação dos critérios de inclusão e exclusão aplicados durante a etapa de seleção, resultando no conjunto final de publicações analisadas.

- **Características Gerais:** apresenta as informações gerais extraídas de cada publicação.

- **Especificações:** reúne as características e informações extraídas sobre os recursos identificados em cada publicação.


## Observações

Este repositório foi preparado para preservar o anonimato dos autores durante o processo de avaliação do manuscrito. Informações que possam identificar os autores, a instituição ou o projeto foram removidas ou anonimizadas.

Após a conclusão do processo de avaliação, este repositório poderá ser atualizado com informações complementares sobre o estudo.
